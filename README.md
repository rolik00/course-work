# Курсовая работа 2 курс 

__"Разработка обучающего приложения по алгоритмам для начинающих программистов"__

*Участники:*
1. Маковеева Анастасия (frontend-разработка)
2. Родина Ольга (backend-разработка)

*Цель курсовой работы:*

Создание десктопного обучающего приложения по алгоритмам для начинающих программистов на Java. Приложение должно предлагать простой и доступный способ изучения базовых тем по алгоритмам и структурам данных, используя теоретические блоки и тесты, проверяющие усвоение материала. Также должен быть финальный тест, проверяющий знания по всем изученным темам, он должен быть доступен только при прохождении всех промежуточных тестов не ниже, чем на 75%.
