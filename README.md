# Курсовая работа 2 курс 

## "Разработка обучающего приложения по алгоритмам для начинающих программистов"

__Участники:__
1. Маковеева Анастасия (frontend-разработка)
2. Родина Ольга (backend-разработка)

__Цель курсовой работы:__

Создание десктопного обучающего приложения по алгоритмам на Java для начинающих программистов. Приложение предоставляет доступный способ изучения базовых тем по алгоритмам и структурам данных с использованием теоретических блоков и тестов для проверки усвоения материала.

__Структура приложения:__  
- Всего 6 разделов, из которых 5 содержат теорию и промежуточный тест, а шестой — итоговый финальный тест.  
- Темы разделов:  
  1. _Основы:_ вычислительная сложность алгоритмов, алгоритмы поиска, поиск подстроки, алгоритм Евклида, решето Эратосфена.  
  2. _Сортировки:_ сортировка пузырьком, вставками, выбором, слиянием, быстрая сортировка.  
  3. _Структуры данных:_ массивы (статические и динамические), связный список, стек, очередь, дерево, куча, хэш-таблица.  
  4. _Графы:_ понятие графа, его элементы, способы представления, основные алгоритмы на графах.  
  5. _Алгоритмические парадигмы:_ рекурсия, метод «грубой силы», «разделяй и властвуй», динамическое программирование, жадные алгоритмы.  
  6. _Финальный тест:_ проверка знаний по всем темам.

__Функциональность:__  
- Финальный тест доступен только после прохождения всех промежуточных тестов с результатом _не менее 75%_ по каждому.  
- Раздел «Статистика» для просмотра результатов по всем 6 тестам.

__Типы вопросов в тестах__ 
1. _Тестовые (множественный выбор):_ выбрать все верные варианты из предложенных.
2. _Открытые:_ вписать правильный ответ в текстовое поле.
3. _Пропуск слова:_ выбрать подходящие слова из списка и вставить на место пропусков в тексте.
4. _Сопоставление:_ установить соответствие между элементами двух колонок.
